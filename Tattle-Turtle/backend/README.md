# Backend for Speech-to-Text and Text-to-Speech

## Setup

1. Install dependencies:
   npm install express cors dotenv axios

2. Set your Google API key in `.env`:
   GOOGLE_API_KEY=your_google_api_key_here

3. Start the server:
   node server.js
